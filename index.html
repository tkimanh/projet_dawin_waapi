WAAPI (Web Animation API)

Qu'est ce que c'est ?

C'est une nouvelle API Javascript réalisée par des ingénieurs de Mozilla et Google. Elle permet de réaliser du contenu animé controlable sur des formes et SVG, comme on le ferait en CSS. A la différence que l'API a été conçu pour faire un pont entre les animations et transitions css, et les animations javascript. Cette différence permet d'utiliser les propriétés données par le css (avec des noms plus ou moins semblables) et les valeurs dynamiques par le javascript.

WAAPI (Web Animation API)

Forces et Faiblesses

  • Documentation en évolution avec peu de tutoriels sur le sujet pour le moment
  • Existence de librairies Javascript offrant plus de choses pré-faites (ex : GSAP, Animate.js)
  • Usage adapté pour les animations complexes contrairement au CSS
  • Possibilité de contrôler les animations avec des méthodes (play, stop, finish, cancel)
  • Permet de se passer de librairies en utilisant des fonctions natives du navigateur, offrant un gain de performance

De quoi a-t-on besoin pour l'utiliser ?

Pour utiliser l'API, il suffit d'avoir un éditeur de texte pour écrire et éditer du code HTML, CSS et Javascript et de comprendre les propriétés permettant d'animer une forme ou un svg.

Code correspondant : 
const monElement = document.querySelector('.exemple-texte-anime')
    .animate({
        transform: ['translate3d(0,-100%,0)'],
    }, {
        direction: 'alternate',
        duration: 500,
        iterations: Infinity,
    });
